PREREQUISITES:
- Java JDK 17 or higher installed
- MySQL Server installed and running

STEP 1: DATABASE SETUP
1. Open your MySQL client (Workbench or Shell).
2. Create a database named 'notes_db'.
3. Import the provided 'notes_db.sql' file into 'notes_db' to set up the tables.
*(Note: Ensure your local MySQL credentials match the application properties)*

STEP 2: RUN THE APPLICATION
1. Open Command Prompt or Terminal inside this project folder.
2. Run the executable JAR using the following command:
   java -jar NoteHub_Application.jar

STEP 3: ACCESS THE APPLICATION
1. Once the terminal shows that Spring Boot has started successfully, open your web browser.
2. Navigate to: http://localhost:8080
3. The React frontend interface and Java backend will both be active on this port.
